Описание prompt

console.log

                            console.log("Some text");

Результат в консоли: "Some text"

                            let a=5;
let b="lizards";
console.log(a," "+b);

Результат в консоли: "5 lizards"

alert()

                            alert("");

Результат: вылезет пустое окно с кнопкой "ОК"

                            let a=5;
let b="lizards";
alert(a+" "+b);

Результат: вылезет уродливое окно с сообщением "5 lizards" и кнопочкой ОК

prompt()

                            prompt("");

Результат: вылезет пустое окно с полем для ввода текста и кнопками "ОК" и "Отмена"

                            let a=5;
let b="lizards";
prompt(a+" "+b);

Результат: вылезет уродливое окно с сообщением "5 lizards", полем для ввода текста и кнопками "ОК" и "Отмена"

                                let a=5;
let b="lizards";
prompt(a+" "+b,"DRAGONS");

Результат: вылезет уродливое окно с сообщением "5 lizards", полем для ввода текста, в котором находится текст по умолчанию "DRAGONS", и кнопками "ОК" и "Отмена"

                            const name_of_const=prompt("type something","DRAGONS");
console.log(name_of_const);

Результат: вылезет уродливое окно с сообщением "type something", полем для ввода текста, в котором находится текст по умолчанию "DRAGONS", и кнопками "ОК" и "Отмена". Далее возможны три сценария.

  • Если вы просто нажмете ОК, то в консоль выведется "DRAGONS".
  • Если же что-то ввели в текстовое поле, то оно отобразится в консоли.
  • Если нажмете "Отмена" при любом введенном тексте, в консоли будет NULL
